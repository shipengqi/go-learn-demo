## Go语言的主要特性
- 编译型语言，静态类型语言
- 自动回收垃圾
- Go 有自己原生的并发编程模型
- 函数式编程
- 基于消息传递的通信方式
- 丰富实用的内置数据类型
- 丰富的标准库
- `defer`机制
- 反射(`reflect`)
- 高性能`HTTP Server`
- 工程管理
- 编程规范
